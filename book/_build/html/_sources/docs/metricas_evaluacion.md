# Métricas de evaluación para la comparación de modelos PINNs

Con el objetivo de evaluar y comparar los resultados obtenidos al resolver el problema inverso, se emplearan métricas estadísticas ampliamente utilizadas en el análisis de modelos. En particular, se utilizaran el criterio de información de Akaike (AIC), el criterio de información Bayesiano (BIC) y el coeficiente de determinación ($R^2$), las cuales permiten cuantificar tanto la calidad del ajuste a los datos experimentales como la complejidad del modelo inferido.

 Mientras que el $R^2$ ofrece una medida directa de la calidad del ajuste a los datos, el AIC y el BIC aportan criterios adicionales para evaluar la eficiencia y robustez de cada modelo.

En el contexto de este proyecto, estas métricas se aplican para comparar diferentes PINNs utilizadas en la estimación de parámetros del modelo de crecimiento microbiano a partir de las series de tiempo del crecimiento de gránulos de kéfir de agua sometidos a distintos pretratamientos de ultrasonido.
